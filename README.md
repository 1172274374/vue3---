
## 安装使用

- 获取项目代码

```bash
git clone https://github.com/1172274374/vue3-youxia.git
```

- 安装依赖

```bash
cd vue3-youxia

npm install

```

- 运行

```bash
npm run serve
```

- 打包

```bash
npm run build
```
